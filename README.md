# An application/portfolio for The European Correspondent's Data Journalist Opening

*hosted [here](https://tec-application.txtos.eu/#ds_need)*

## Description 
A static site inspired by the look of a NYT physical cover. Covers my skills and interest in the position and gives two visualization showcases. 

## Technologies
- front-end: vanilla HTML and CSS. 
- visualizations: D3.js. 
- data analysis (e.g. scatterplot's line interpolation, data cleaning, data normalization): Python. Not included in the repository.
